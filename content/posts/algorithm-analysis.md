# 短视频算法逻辑拆解

理解平台算法是内容创作者的必修课。本文将深入解析主流短视频平台的推荐算法逻辑。

## 1. 核心指标

平台主要关注以下指标：

| 指标 | 权重 | 说明 |
|------|------|------|
| 完播率 | 高 | 用户是否看完视频 |
| 互动率 | 高 | 点赞、评论、分享 |
| 关注转化 | 中 | 是否关注账号 |
| 复看率 | 中 | 是否重复观看 |

## 2. 推荐流程

```
发布 → 初始流量池 → 数据评估 → 扩大推荐 / 停止推荐
```

### 初始流量池

每个新发布的视频都会获得 200-500 的初始曝光。

### 数据评估

系统会根据以下公式计算推荐分数：

```
分数 = 完播率 × 0.4 + 互动率 × 0.3 + 关注转化 × 0.2 + 复看率 × 0.1
```

## 3. 优化建议

1. **前 3 秒定生死**：开头必须抓住注意力
2. **引导互动**：在视频中自然引导点赞评论
3. **内容垂直**：保持账号内容的一致性
