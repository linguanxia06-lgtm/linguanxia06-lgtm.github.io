# 个人门户网站产品需求文档 (PRD)

## 1. 视觉风格规范 (Visual Guidelines)

- **色彩方案**:
  - 背景: `#FFFFFF` (纯白)
  - 主文字: `#1D1D1F` (深黑)
  - 次要文字: `#86868B` (浅灰)
- **圆角规范**: 苹果标准圆角 (如 `rounded-2xl` 或 `12px/20px`)。
- **动效要求**: 使用 `Framer Motion` 实现丝滑的人场淡入效果，严禁过度晃动的动画。
- **主题模式**: 仅支持浅色模式，专注于极致的纯白苹果风。

## 2. 部署与性能 (Deployment)

- **静态化**: 必须支持 `next export` 静态导出。
- **部署平台**: GitHub Pages。
- **自动化**: 包含 GitHub Actions 配置文件，实现 Push 即部署。

## 3. 页面结构与路由 (Routing)

| 路由             | 页面名称  | 核心逻辑                             |
| :------------- | :---- | :------------------------------- |
| `/`            | 首页    | 包含 Hero Section + 精选作品预览 (Top 3) |
| `/work`        | 作品全集  | 所有视频/项目案例的过滤与展示 (支持 Tag 过滤)      |
| `/blog`        | 思考记录  | Markdown 文章列表，按时间倒序排列，支持标签分类      |
| `/blog/[slug]` | 文章详情  | 渲染 MDX/Markdown 内容，支持代码高亮        |
| `/about`       | 关于/简历 | 结构化展示个人履历、技能及联系方式                |

## 4. 数据 Schema 预览 (Data Definition)

### 4.1 作品数据结构 (Project)

```json
{
  "project": {
    "id": "1",
    "title": "项目名称",
    "category": "AI-Strategy",
    "cover": "/images/work1.jpg",
    "link": "https://v.douyin.com/xxx",
    "metrics": "100W+播放",
    "tags": ["Midjourney", "HeyGen", "编剧"]
  }
}
```

### 4.2 文章数据结构 (Post)

```json
{
  "post": {
    "title": "文章标题",
    "date": "2024-XX-XX",
    "tags": ["AI工具", "策划"],
    "summary": "内容摘要",
    "content": "Markdown 正文"
  }

```

## 5. 核心交互需求

1. **外链跳转**: 所有作品卡片点击后在新标签页打开外部链接。
2. **标签过滤**: 在 /work 页面通过 Tag 实时筛选作品，需具备平滑的重排动画。
3. **简历下载**: Hero 区域"下载简历"按钮，点击后在新标签页打开 `/resume.pdf`。

## 6. 作品集规模

- **预计数量**: 10-20 个作品
- **展示方式**: 网格排列，使用"加载更多"按钮或简单分页
- **无需复杂瀑布流**: 保持简洁的响应式网格布局

## 7. About 页面结构

About 页面包含以下三个板块：

| 板块 | 内容描述 |
|------|----------|
| **个人简介** | 简短的自我介绍，突出核心定位 |
| **工作经历** | 时间轴形式展示职业经历 |
| **AI工具栈** | 标签云形式展示熟练使用的 AI 工具 |

## 8. 响应式设计要求

| 设备类型 | 作品网格 | 导航栏 |
|----------|----------|--------|
| 桌面端 (≥1024px) | 3 列 | 水平导航 |
| 平板端 (≥768px) | 2 列 | 水平导航 |
| 移动端 (<768px) | 1 列纵向 | Hamburger Menu (汉堡菜单) |

## 9. SEO 配置

- **Meta 标签**: 基础 title、description、keywords
- **Open Graph**: 标题格式为 `[名字] | 编剧 & AI 策划`
- **无需复杂 SEO**: 保持基础配置即可

## 10. 导航栏链接

- **GitHub Icon**: 链接指向个人 GitHub 主页
- **导航项**: Works、Blog、About
